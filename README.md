# Heart Disease Prediction with Predictive Analytics and Machine Learning

## About the Project
This project focuses on building a predictive analytics and machine learning pipeline for heart disease prediction using healthcare data. The objective of the project is to analyze patient health-related attributes and predict the likelihood of heart disease using multiple machine learning algorithms.
The workflow includes data preprocessing, handling missing values, feature engineering, data visualization, model training, evaluation, and comparison of different machine learning techniques.
Several classification models, clustering techniques, and dimensionality reduction methods were implemented to better understand the dataset and improve predictive performance.
The project was developed using Python in Google Colab with the help of libraries such as Pandas, NumPy, Scikit-learn, TensorFlow, Matplotlib, and Seaborn.

## Dataset Information
The dataset used in this project was taken from Kaggle and contains medical and clinical attributes related to heart disease diagnosis.
Some important features include:
- Age
- Chest pain type
- Cholesterol level
- Resting blood pressure
- Maximum heart rate
- Blood sugar level
- ECG results

The target variable represents the presence or absence of heart disease.

## Project Workflow
1. Data Cleaning and Preprocessing
2. Missing Value Handling
3. Feature Encoding
4. Data Scaling and Normalization
5. Exploratory Data Analysis (EDA)
6. Model Training and Testing
7. Performance Evaluation
8. Clustering and PCA Analysis
9. Final Model Comparison
   
## Machine Learning Models Used
- Logistic Regression
- Decision Tree Classifier
- Random Forest Classifier
- K-Nearest Neighbors (KNN)
- Naive Bayes
- Artificial Neural Network (ANN)
- K-Means Clustering
- Principal Component Analysis (PCA)

## Results and Findings
Different machine learning models were trained and evaluated using metrics such as Accuracy, Precision, Recall, and F1-score.
Among the implemented models, Random Forest achieved strong and stable predictive performance due to its ensemble learning capability and ability to reduce overfitting. Artificial Neural Network (ANN) also produced competitive results by learning complex non-linear relationships within the dataset.
The project also included feature importance analysis, clustering using K-Means, and dimensionality reduction using PCA to better understand patterns within the healthcare data.
Features such as chest pain type, cholesterol level, resting blood pressure, and maximum heart rate showed significant impact on prediction results.
Overall, the project demonstrates how machine learning and predictive analytics techniques can be applied to healthcare datasets for disease prediction and data-driven insights.
